## Install
Install these packages globally.
```
npm install -g http-server
npm install -g webpack
npm install -g babel-loader
npm install -g babel-core
npm install -g babel-preset-es2015
```

Run npm install to link them.
```
npm install
```

## Run
```
http-server
```

## Build
```
webpack
```
